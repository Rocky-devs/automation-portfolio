# Dental Clinic AI Receptionist + Appointment Automation

## Overview
Full AI receptionist and appointment automation system for dental clinics.
Handles both online bookings and after-hours calls automatically.

[Watch Demo Video](https://youtu.be/2kE0Pue_Cjk)

## Features
- AI voice receptionist answers after-hours calls (Retell AI)
- Online appointment booking via form (Tally)
- Instant clinic notifications via Gmail + Telegram
- Patient confirmation email auto-sent
- Single workflow handles both form and voice inputs
- WhatsApp / SMS integration available

## Tech Stack
- **Retell AI** — voice receptionist
- **n8n** — workflow automation
- **Railway** — cloud deployment
- **Tally** — online booking form
- **Gmail + Telegram** — notifications

## How It Works
1. Patient fills online form OR calls after hours
2. n8n detects source (form vs voice call)
3. Clinic receives instant Gmail + Telegram notification
4. Patient receives confirmation email (form bookings)
